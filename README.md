# learning-vue

## Project setup
```
yarn install
```

### Start server
```
yarn run server:start
```

### Start app
```
yarn run serve
```

### Compiles and minifies app for production
```
yarn run build
```

### Run end-to-end tests
```
yarn run test:e2e
```

### Run server tests
```
yarn run test:server
```

### Run Vue tests
```
yarn run test:unit
```
